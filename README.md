## WalkingBird
再也不必拿著地圖對方位，實境導航直接告訴你該怎麼走
App Store: <https://itunes.apple.com/tw/app/walking-bird/id1332543881>

## Feature
* 設定起點和終點，實境導航告訴你走路到目的地的最佳路徑
* 儲存常用路徑，下次要用直接點它
* 邀請朋友走同一條路，讓他探索這條未知目的地的路徑


## Libraries
* ARKit
* SceneKit
* CoreLocation
* MapKit
* ARCL
* YXWaveView
* SwiftLint
* Firebase
* Firebase Analytics
* Crashlytics
